# 🎾 Tennis Shop

A modern full-stack e-commerce application for a tennis equipment store, built with Next.js 15, TypeScript, Prisma, and MongoDB.

## 🚀 Features

- User authentication (NextAuth)
- Product catalog
- Product filtering
- Product details page
- Shopping cart
- Order management
- Responsive design
- Admin panel
- MongoDB database with Prisma ORM

## 🛠 Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- DaisyUI
- Prisma ORM
- MongoDB Atlas
- NextAuth
- Vitest

## ⚙️ Installation

```bash
git clone https://github.com/TanuhaD/tennis-shop.git

cd tennis-shop

npm install

npm run dev
```
