# 🎾 Tennis Shop

A modern full-stack e-commerce application for a tennis equipment store, built with Next.js 15, TypeScript, Prisma, and MongoDB.

## 🚀 Features

- User authentication (NextAuth)
- Product catalog
- Product filtering
- Product details page
- Shopping cart
- Order management
- Responsive design
- Admin panel
- MongoDB database with Prisma ORM

---

## 🛠 Tech Stack

### Frontend

- Next.js 15 (App Router)
- React
- TypeScript
- Tailwind CSS
- DaisyUI

### Backend

- Next.js Route Handlers
- Prisma ORM
- MongoDB Atlas
- NextAuth

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/TanuhaD/tennis-shop.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file and configure:

- MongoDB connection string
- NextAuth configuration

Run the development server

```bash
npm run dev
```
